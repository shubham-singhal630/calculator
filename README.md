# 🧮 Calcify – Modern Calculator

A sleek and feature-rich calculator built using **HTML, CSS, and JavaScript**. Calcify combines a modern user interface with advanced functionality such as scientific calculations, memory operations, calculation history, and an integrated unit converter.

---

## ✨ Features

### 🔢 Standard Calculator

* Basic arithmetic operations:

  * Addition (+)
  * Subtraction (−)
  * Multiplication (×)
  * Division (÷)
* Percentage calculations
* Positive/Negative toggle
* Decimal number support
* Clear All (AC)

### 🧪 Scientific Calculator

Supports various scientific operations including:

* Sine (`sin`)
* Cosine (`cos`)
* Tangent (`tan`)
* Logarithm (`log`)
* Natural Log (`ln`)
* Square Root (`√`)
* Square (`x²`)
* Cube (`x³`)
* Power (`xʸ`)
* Reciprocal (`1/x`)
* Absolute Value (`|x|`)
* Factorial (`n!`)
* 10 to the Power (`10ˣ`)
* Mathematical Constants:

  * π (Pi)
  * e (Euler's Number)

### 📐 Degree / Radian Mode

* Toggle between **DEG** and **RAD** modes for trigonometric calculations.

### 🧠 Memory Functions

* MC – Memory Clear
* MR – Memory Recall
* M+ – Memory Add
* M− – Memory Subtract

### 🕒 Calculation History

* Stores recent calculations.
* Reload previous results instantly.
* Clear history with a single click.
* Supports up to 50 history records.

### 📏 Unit Converter

Integrated converter supporting:

#### Length

* Meter
* Kilometer
* Centimeter
* Millimeter
* Inch
* Foot
* Yard
* Mile

#### Weight

* Kilogram
* Gram
* Pound
* Ounce
* Ton

#### Temperature

* Celsius
* Fahrenheit
* Kelvin

#### Area

* Square Meter
* Square Kilometer
* Square Centimeter
* Square Foot
* Square Inch
* Acre
* Hectare

#### Speed

* Meter/Second
* Kilometer/Hour
* Miles/Hour
* Knot

### ⌨️ Keyboard Support

Use your keyboard for faster calculations:

| Key       | Action            |
| --------- | ----------------- |
| 0–9       | Number Input      |
| +         | Addition          |
| -         | Subtraction       |
| *         | Multiplication    |
| /         | Division          |
| Enter / = | Calculate         |
| Backspace | Delete Last Digit |
| Escape    | Clear All         |
| %         | Percentage        |

### 📱 Responsive Design

* Optimized for desktops, tablets, and mobile devices.
* Adaptive layout for smaller screens.
